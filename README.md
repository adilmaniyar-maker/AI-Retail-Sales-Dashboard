# AI-Powered Retail Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data using Python and Power BI to
identify trends, track key performance indicators (KPIs), and generate
actionable business insights.

## Tools & Technologies

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Jupyter Notebook
-   Power BI
-   Excel

## Dashboard Features

-   Total Sales KPI
-   Total Profit KPI
-   Total Orders KPI
-   Sales by Category
-   Sales by Region
-   Monthly Sales Trend
-   Top 10 Products by Sales
-   Profit by Sub-Category
-   Interactive filters for Region, Category, and Segment

## Key Insights

-   Technology products contribute significantly to sales.
-   Regional analysis helps identify strong and weak markets.
-   Monthly trends reveal seasonal demand patterns.
-   Product-level analysis highlights top-performing items.

## Repository Contents

-   `Retail_Sales_Analysis.ipynb` -- Python analysis notebook
-   `AI_Retail_Sales_Dashboard.pbix` -- Power BI dashboard
-   `dashboard.png` -- Dashboard screenshot
-   `requirements.txt` -- Python dependencies
